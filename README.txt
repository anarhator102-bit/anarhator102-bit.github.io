
1. Установи Node.js

2. Открой папку проекта в VS Code

3. В терминале:

npm install

4. Потом:

npm start

5. Открой:
http://localhost:3000

=========================

КАК ЗАЛИТЬ В ИНТЕРНЕТ

1. Залей server.js package.json tracks.json на GitHub

2. Зайди:
https://render.com

3. Создай Web Service

4. Build Command:
npm install

5. Start Command:
node server.js

6. После deploy Render даст ссылку.

7. В index.html замени:

https://YOUR_RENDER_SERVER.onrender.com

на ссылку Render.

Теперь:
- треки сохраняются
- музыка сохраняется
- обложки сохраняются
- всё видно всем
