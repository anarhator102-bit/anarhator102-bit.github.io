
1. Открой папку проекта в Visual Studio Code

2. Открой терминал

3. Установи зависимости:

npm install

4. Запусти сервер:

npm start

5. Открой:

http://localhost:3000

Теперь:
- треки сохраняются в папку uploads
- обложки сохраняются
- музыка работает после перезагрузки сайта
- данные сохраняются в tracks.json
